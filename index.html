A "Sopa de Letrinhas" de Comandos

A1 e A2 (Bobina): É o eletroímã. Recebe o sinal de comando (ex: 24VDC do CLP) para atracar.

L1, L2, L3 (Line): Entrada de potência principal. Fases vindas do disjuntor geral.

T1, T2, T3 (Terminal): Saída de potência. Vai direto para a carga pesada.

NA / NO (Normalmente Aberto): Terminação 3 e 4. Quando a bobina atinge tensão, ele fecha. (Selo, Luz verde).

NF / NC (Normalmente Fechado): Terminação 1 e 2. Quando a bobina atinge tensão, ele abre. (Intertravamento, Luz vermelha).

Catálogo Visual de Painel

Contator de Potência (WEG)

A chave principal de força. A bobina (A1/A2) aciona os contatos principais (L/T) para suportar altas correntes.

Bloco Auxiliar

Multiplica os contatos disponíveis para enviar feedback ao CLP, ligar lâmpadas ou fazer travas.

Relé de Interface

Protege as saídas delicadas do CLP. O CLP aciona a bobina deste relé, e o relé aciona o contator maior.

Esquema Prático

Distribuição onde blocos auxiliares enviam sinais para os sinaleiros do painel (Verde = NO, Vermelho = NC).
